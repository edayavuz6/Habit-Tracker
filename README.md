📌 About the Project
This is a modern, functional, and aesthetically pleasing habit-tracking web application. The primary goal of this project was to handle complex data logic using Vanilla JavaScript while providing a high-quality user experience (UX).

LIVE CODE : https://edayavuz6.github.io/Habit-Tracker/

✨ Key Features
Dual View System: Seamless toggle between Daily List view and Weekly Matrix view.

Data Persistence: Integrated localStorage ensures user data remains safe even after page reloads.

Advanced Analytics: * Real-time weekly success percentage calculation.

"Longest Streak" tracking to boost user motivation.

Visual progress bars and completion counters.

Theming: Professional Dark Mode support managed via CSS Variables and saved to user preferences.

Dynamic Visuals: Custom color selection for each habit and confetti effects upon completion.

🛠️ Technical Deep Dive
HTML5 & CSS3: Responsive layouts using modern Flexbox/Grid and smooth UI transitions.

JavaScript (ES6+): * DOM Manipulation: Efficiently rendering and updating elements without external libraries.

Logic & Algorithms: Complex percentage calculations and date-based data filtering.

State Management: Centralized handling of application data to ensure UI-Sync.
